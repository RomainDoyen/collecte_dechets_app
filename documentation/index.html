Technologies utilisées

Application mobile

Flutter Dart Material 3
  • Flutter — Framework multiplateforme (Android / iOS)
  • Dart — Langage typé et performant
  • Material 3 — Design system moderne
  • Table Calendar — Calendrier interactif
  • WorkManager — Tâches en arrière-plan
  • Android Alarm Manager — Alarmes exactes natives

Backend & Services

Firebase Firestore FCM
  • Firebase Firestore — Base de données cloud NoSQL
  • Firebase Cloud Messaging — Notifications push
  • Flutter Local Notifications — Notifications locales programmées
  • SharedPreferences — Stockage local de données
  • Timezone — Gestion des fuseaux horaires (Indian/Reunion)
